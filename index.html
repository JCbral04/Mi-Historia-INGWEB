Capítulo 1

El Primer Encuentro

Todo comenzó con una simple pregunta: ¿Cómo se construyen las páginas web? Lo que parecía un misterio inalcanzable se convirtió en una puerta abierta a un mundo de posibilidades infinitas.

Descubrí que detrás de cada sitio web hay una historia de estructura, diseño y lógica. HTML era el esqueleto, CSS la piel y JavaScript el alma que todo lo anima.

index.html 

<!DOCTYPE html>
<html lang="es">
  <body>
    <h1>Hola Mundo</h1>
    <p>Mi primera página</p>
  </body>
</html>

Resultado:

Hola Mundo

Mi primera página
